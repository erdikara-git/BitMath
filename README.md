# BitMath

BitMath is a simple and fast math practice app designed to improve basic arithmetic skills.  
It provides clean UI, instant feedback, and multiple difficulty levels.

## Features

- Addition, subtraction, multiplication, division
- Increasing difficulty levels
- Simple and clear interface
- Fast gameplay flow
- Ideal for daily practice
