# BitMath

Bitmath is a learning app for practicing basic arithmetic under time pressure. The app offers two game types. In the first type the player enters the result manually. In the second type the player chooses the correct answer from four options. Three difficulty levels adjust the speed and complexity of the tasks. The app is available in German English and Turkish. Players can earn coins and a highscore table is planned.

Features
Basic arithmetic training
Two game types
Three difficulty levels
Three languages
Time based challenges
Coin rewards system planned
Highscore table planned
