# BitMath

A simple calculator app for basic arithmetic operations with a clean and intuitive interface.  
Designed for quick addition, subtraction, multiplication, and division.

## Features

• Input of two numbers  
• Automatic calculation of all four basic operations  
• Separate result display for each operation  
• Error handling for invalid input  
• Easy and lightweight UI

## Installation

1. Clone the repository  
   `git clone https://github.com/erdikara-git/BitMath.git`

2. Open the project  
   Launch Android Studio and select the project folder

3. Run the app  
   Start it on an emulator or on a physical Android device
